HDLStudio

A modern, self-contained IDE for Verilog, SystemVerilog and VHDL design — with simulation, synthesis, place & route, and waveform viewing.

Get Started Docs GitHub

What is HDLStudio?

HDLStudio is an open-source integrated development environment for digital hardware design. It bundles all the essential tools — Icarus Verilog, GTKWave, Graphviz — so you can start designing without hunting for dependencies.

Key Features

✏

Code Editor

Syntax highlighting, autocompletion, snippets and hover docs for Verilog, SystemVerilog and VHDL.

⚙

Dual Simulator

Compile and run with Icarus Verilog (bundled) or Verilator (via WSL). Switch at any time.

📊

Waveform Viewer

Launch GTKWave directly from the IDE to inspect VCD, FST, LXT and GHW traces.

🏭

Synthesis

Yosys-based synthesis with BLIF, JSON, Verilog and EDIF output formats.

🔌

Place & Route

nextpnr-ice40 for Lattice iCE40 FPGAs with structured metrics reports.

📦

Extensible

Python-based plugin system lets you add toolbar actions, menu items, bottom-panel tabs and more.

Quick Start

All bundled tools are included. No separate downloads needed for basic use.
bashgit clone https://github.com/josped385/HDLStudio.git
cd HDLStudio
pip install -r requirements.txt
python main.py

Select an HDL file in the file explorer, pick your simulator from the toolbar, and press F5 to compile.
