Features at a Glance

  • Code editor with syntax highlighting for Verilog, SystemVerilog, VHDL
  • Simulation with Icarus Verilog (bundled) or Verilator (WSL2)
  • GTKWave waveform viewer (bundled)
  • Yosys-based synthesis (BLIF, JSON, Verilog, EDIF)
  • Gate-level schematic viewer (Yosys + Graphviz)
  • Lattice iCE40 place & route (nextpnr-ice40)
  • Python extension/plugin system
  • Dark and light themes
  • Settings dialog with editor and extension configuration

Acknowledgements

HDLStudio uses the yowasp Wasm packages for Yosys and nextpnr-ice40, making synthesis and PnR available without requiring MSYS2/MinGW builds on Windows.

Thanks to the open-source hardware community for creating and maintaining the underlying tools.
