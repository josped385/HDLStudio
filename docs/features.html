Features

A comprehensive tour of everything HDLStudio can do.

Editor

  • ✓ Syntax highlighting for Verilog, SystemVerilog and VHDL
  • ✓ Autocompletion and code snippets
  • ✓ Hover documentation (port types, signal widths)
  • ✓ Find and replace across all open files
  • ✓ Configurable font family, size, tab width and word wrap
  • ✓ Multiple tabs with unsaved-changes indicator
  • ✓ Error markers on compilation failure (clickable error links in console)

Project Management

  • ✓ File explorer with tree view (expand/collapse, right-click actions)
  • ✓ Open folder as project root
  • ✓ Auto-detection of module and testbench files (by filename convention)
  • ✓ Manual override via Module / Testbench dropdowns in the toolbar
  • ✓ Module hierarchy tree and signal/port browser

Simulation

  • ✓ Compile and run with Icarus Verilog (local copy in tools/iverilog/)
  • ✓ Compile and run with Verilator (via WSL2 + Ubuntu, optional)
  • ✓ Simulator selector in the toolbar — switch without restart
  • ✓ Automatic testbench generation from any module/entity
  • ✓ Detects clock ports and generates clock stimuli
  • ✓ Open GTKWave with the generated VCD/FST at the press of a key (F7)

Synthesis

  • ✓ Yosys-based synthesis (installed via yowasp-yosys pip package)
  • ✓ Output formats: BLIF, JSON, Verilog, EDIF
  • ✓ Synthesize from right-click menu or Tools menu

Schematic Viewer

  • ✓ Gate-level schematic generated via Yosys show + Graphviz dot
  • ✓ Can start from an HDL file or from a previous synthesis BLIF
  • ✓ SVG output displayed in a dialog, fallback to raw DOT text

Place & Route

  • ✓ nextpnr-ice40 integration for Lattice iCE40 FPGAs (via yowasp-nextpnr-ice40)
  • ✓ Automatic detection of last synthesis JSON output
  • ✓ Structured metrics dialog (logic cells, I/O, delay lines)
  • ✓ Full log capture and display

UI & Customisation

  • ✓ Dark and light themes (toggle with Ctrl+T)
  • ✓ Activity bar with file explorer, hierarchy, signals, templates, git
  • ✓ Bottom panel with Console, Terminal, Log and History tabs
  • ✓ Settings dialog (Editor font/size/tabs, auto-save, extensions)
  • ✓ Python plugin system (extensions/ directory)
