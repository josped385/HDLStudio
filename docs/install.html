Installation

Running HDLStudio on Windows 10 / 11.

📥 Download the standalone installer

⬇ Download HDLStudio_Setup_v0.1.0.exe

~100 MB — Windows 10/11 64-bit — No external dependencies

From Source

Alternatively, clone the repository and run from Python:

Prerequisites

DependencyRequired?Notes
Python 3.10+Yes64-bit recommended
pipYesIncluded with Python
GitFor updatesgit clone the repo
WSL2 + UbuntuOptionalFor Verilator support

Step-by-step

  1. Clone the repository 
    bashgit clone https://github.com/josped385/HDLStudio.git
cd HDLStudio
  2. Create a virtual environment (recommended) 
    bashpython -m venv venv
.\venv\Scripts\activate
  3. Install dependencies 
    bashpip install -r requirements.txt
  4. Launch the IDE 
    bashpython main.py

Icarus Verilog, GTKWave and Graphviz must be placed in tools/ locally — they are not distributed in the repository due to their size. Yosys and nextpnr-ice40 are installed via pip (yowasp-yosys, yowasp-nextpnr-ice40).

Optional: Verilator (WSL2)

Verilator does not run natively on Windows. It must be used through WSL2.
  1. Install WSL2 — run wsl --install -d Ubuntu in an admin PowerShell
  2. Install build tools in Ubuntu 
    bashsudo apt update
sudo apt install build-essential git help2man libgoogle-perftools-dev
sudo apt install numactl perl python3 python3-venv
sudo apt install autoconf flex bison ccache libfl-dev libgd-dev
  3. Build and install Verilator 
    bashgit clone https://github.com/verilator/verilator --depth 1
cd verilator
autoconf
./configure --prefix=/usr/local
make -j$(nproc)
sudo make install

HDLStudio detects Verilator automatically by running wsl verilator --version. Select "Verilator" from the simulator dropdown in the toolbar.

Optional: PyInstaller Build

To package HDLStudio into a standalone .exe:

bashpip install pyinstaller
pyinstaller --clean HDLStudio.spec

The output appears in dist/HDLStudio/. A standalone installer can be built with Inno Setup 6+ using installer.iss.
