Usage Guide

How to use HDLStudio for your hardware design workflow.

Opening a Project

Click File > Open Folder (or Ctrl+K Ctrl+O) and select a directory containing your HDL source files. The file explorer in the activity bar will show all files.

Editing HDL

Double-click a file in the explorer to open it in a new tab. The editor provides:

  • Syntax highlighting for .v, .sv, .vhdl, .vhd
  • Autocompletion and code snippets
  • Hover documentation showing port directions and signal widths
  • Find & Replace (Ctrl+F) across the current file or all open files

Simulation

Selecting a Simulator

The toolbar has a Simulator: dropdown that lets you switch between Icarus Verilog (default, bundled) and Verilator (requires WSL2).

Compiling

  • F5 — Compile the current file
  • F6 — Compile all files in the project

Icarus produces a .vvp file. Verilator produces an executable binary in build/verilator_obj/.

Running Simulation

  • F9 — Run the compiled simulation (writes a .vcd trace)

Viewing Waveforms

  • F7 — Open the generated VCD file in GTKWave

Auto-generating a Testbench

Right-click any module in the hierarchy panel and select Generate Testbench, or use Tools > Generate Testbench. HDLStudio will parse the module ports, detect clock signals (by name conventions), and create a testbench skeleton with $dumpvars and clock stimuli.

Synthesis

Right-click a file in the explorer and select Synthesize, or go to Tools > Synthesize (Ctrl+Shift+Y). Choose the output format:

ExtensionYosys ScriptTypical Use
.blifsynth; write_blifGeneral logic representation
.jsonsynth_ice40 -jsonPreparing for nextpnr-ice40
.vsynth; write_verilogPost-synthesis netlist
.edifsynth; write_edifThird-party EDA interchange

Schematic View

After synthesis, right-click the output file and select Show Schematic, or right-click any HDL file and select Show Schematic from HDL. HDLStudio runs yosys show -format dot and converts the DOT output to an SVG via Graphviz. The schematic is displayed in a dialog window.

Place & Route

Right-click a synthesized JSON file and select Place & Route (Ctrl+Shift+P), or use the toolbar button / Tools > Place & Route.

HDLStudio runs yowasp-nextpnr-ice40 with your JSON, producing an .asc bitstream file, plus a report with logic cell usage, I/O block counts and delay line data.

A summary dialog displays structured metrics. The complete log is available in the PnR output window.
